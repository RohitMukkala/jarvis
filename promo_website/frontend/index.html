Try These Commands

Just start by saying "Jarvis..."

"Open Google"

Launches google.com in your browser.

"Open YouTube"

Launches youtube.com.

"What's the time?"

Tells you the current time.

"Tell me a joke"

Shares a random joke.

"Play music"

Plays the first song in your music directory.

"Show me the latest news"

Fetches and reads top news headlines.

"Weather in [City Name]"

Provides the current weather for a city.

"Where is [Location]"

Finds a location on Google Maps.

"Goodbye"

Shuts down the Jarvis assistant.

More Than Just an Assistant

Jarvis started as a personal project to explore the power of Python for desktop automation. It combines a voice-activated command interface with an intelligent chatbot for conversation.

It also includes experimental features like a heart disease prediction model, showcasing how machine learning can be integrated into a daily-use desktop application.
